# XGram - B2B Construction Materials Marketplace

A premium, clean, and responsive B2B marketplace for construction materials.

## Features
- **Modern UI/UX**: Light theme, professional color palette, smooth animations.
- **Responsive Design**: Mobile-friendly with hamburger menu and collapsible filters.
- **Product Catalog**: Dynamic product grid with "Load More" functionality.
- **Hero Slider**: Smooth auto-sliding carousel.
- **Filtering**: Collapsible categories and sorting options.
- **Product Details**: Dedicated pages for product specifications.
- **Static Pages**: About Us, Contact Us, Terms, Privacy.

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- CSS Modules (Vanilla CSS)
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
